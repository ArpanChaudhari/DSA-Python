## 🎯 About The Repository

This repository serves as a personal log of my journey in mastering Data Structures and Algorithms. The solutions are primarily written in **Python 3**, focusing on clean, optimal, and readable code. 

The problems in this repository are curated from top coding platforms and structured learning sheets, specifically:

- **[Striver's A to Z DSA Sheet](https://takeuforward.org/strivers-a2z-dsa-course/strivers-a2z-dsa-course-sheet-2/)**: A step-by-step complete algorithmic roadmap.
- **[LeetCode 75 Study Plan](https://leetcode.com/studyplan/leetcode-75/)**: A curated list of 75 essential LeetCode questions to master core data structures.
- **Daily LeetCode Practice**: Assorted random problems solved during daily practice.

---

## 🚀 Key Highlights

- **Optimal Solutions:** Every problem contains the most optimal approach (Time & Space complexity) after transitioning from brute-force to better approaches.
- **Pythonic Code:** Leveraging Python's built-in functions and standard data structures (like `collections`, `heapq`, `math`) to write clean and concise code.
- **Well Documented:** Tricky problems include comments explaining the intuition and logic behind the solution.
